# 全局快捷键功能说明

## 功能概述

CommandSelector 支持全局快捷键，可以在任意应用中快速呼出或隐藏程序窗口。

## 默认快捷键

- **默认快捷键**: `Alt + C`

## 使用方法

### 1. 在设置中启用快捷键

1. 打开应用
2. 点击右上角设置图标，进入设置页面
3. 在"全局快捷键"卡片中，确认快捷键开关已启用
4. 点击"应用设置"按钮注册快捷键

### 2. 快捷键操作

- **快捷键触发**: 在任意应用中按下 `Alt + C`
- **显示窗口**: 当窗口隐藏时，按下快捷键会显示窗口并置顶
- **隐藏窗口**: 当窗口显示时，按下快捷键会隐藏窗口

### 3. 自定义快捷键

在设置页面可以自定义快捷键：
- **修饰键**: 选择 Alt、Ctrl、Shift 或组合键
- **字母键**: 选择 A-Z 任意字母
- **应用设置**: 点击"应用设置"保存新快捷键

### 4. 测试快捷键

- 在设置页面点击"测试显示/隐藏"按钮
- 该按钮会模拟快捷键触发窗口切换

## 技术实现

- **后端**: Rust + Windows API (`RegisterHotKey` + `WM_HOTKEY` 消息循环)
- **状态管理**: 使用 `std::sync::Mutex` 确保线程安全
- **消息处理**: 创建隐藏窗口接收 `WM_HOTKEY` 消息，通过 Tauri 事件 `global-hotkey-pressed` 通知前端
- **前端**: Vue 3 + Tauri API 调用，监听 `global-hotkey-pressed` 事件触发窗口切换

## 注意事项

1. **快捷键冲突**: 如果快捷键被其他应用占用，注册会失败
2. **窗口状态**: 快捷键根据窗口可见性自动切换显示/隐藏
3. **置顶逻辑**: 显示窗口时会短暂置顶，避免被其他窗口遮挡
4. **开发环境**: 在开发模式下测试快捷键功能可能不稳定，建议在打包后测试
